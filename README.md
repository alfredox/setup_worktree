# setup_worktree
Setup script for your project after creating a worktree.

## Install

Just copy into your bin directory. For now, it has the usual files for rails:

```
curl -L -o bin/setup_worktree https://raw.githubusercontent.com/alfredox/setup_worktree/refs/heads/main/setup_worktree_rails
chmod +x bin/setup_worktree
```
